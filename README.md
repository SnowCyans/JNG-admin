# adminSystem-main

> 这是一个极简的 vue admin 管理后台。它只包含了 Element UI & axios & iconfont & permission control & lint，这些搭建后台必要的东西。

```bash
# 克隆项目
git clone git@github.com:powderbluen/JNG-admin.git

# 进入项目目录
cd adminSystem-main

# 安装依赖
yarn

# 启动服务
yarn serve
```

## 发布

```bash
# 构建测试环境
yarn run build:stage

# 构建生产环境
yarn run build:prod
```

## 其它

```bash
# 预览发布环境效果
yarn run preview

# 预览发布环境效果 + 静态资源分析
yarn run preview -- --report
```
